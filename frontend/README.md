# SeemantSaathi - Frontend

SeemantSaathi is a simple and easy-to-use health assistant designed for people living in rural areas, specifically in **Bhimtal, Uttarakhand**. 

This folder contains the frontend code for the website, which helps users check their health symptoms, find nearby doctors, and learn basic first aid.

##  Main Features

- **Symptom Checker**: Uses AI to help you understand what your health symptoms might mean.
- **Find Doctors**: Shows a map of hospitals and clinics near Bhimtal.
- **First Aid Guide**: Provides quick life-saving tips for emergencies like snake bites, burns, or cuts.
- **Hindi & English Support**: Everything on the website can be switched between Hindi and English so everyone can understand.
- **Easy Login**: Simple mobile number and OTP login for patients.

##  Folder Structure

- `index.html`: The main home page and login screen.
- `symptoms.html`: The page where you can check your health symptoms.
- `doctors.html`: The page to find doctors and hospitals on a map.
- `style.css`: Contains all the designs and colors for the website.
- `script.js`: The "brain" of the website that handles translations and logic.
- `images/`: A folder for photos and icons.

##  How to Run

Since this is a simple frontend project, you don't need to install anything special.

1. Download or clone this folder.
2. Open `index.html` in any web browser (like Google Chrome or Microsoft Edge).
3. That's it! The website will start working.

##  Built With

- **HTML & CSS**: For the structure and design.
- **JavaScript**: For making the website interactive.
- **Leaflet.js**: For the maps.
- **Ionicons**: For the small icons.
